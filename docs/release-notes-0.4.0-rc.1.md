# Conduit 0.4.0 release candidate 1

Conduit 0.4 establishes the Conduit Goal Specification (CGS) as the portable,
independently versioned contract for goals, clarification decisions, reviews,
evidence, runs, and reports. Conduit Desktop remains the first graphical CGS
client, while `@conduit/runtime` exposes the headless execution boundary.

Component versions in this candidate are:

- Conduit Desktop `0.4.0-rc.1`
- Conduit Runtime `0.4.0-rc.1`
- Conduit Goal Specification `0.1.0`

The existing goal-builder, implementation loop, reviewer routing, evidence
collection, persistence, report viewing/export, providers, and permission UX
remain available through the compatibility migration while stored development
records are converted explicitly.
