# Conduit 0.4.0

Conduit 0.4 establishes the Conduit Goal Specification (CGS) as the portable,
independently versioned contract for goals, clarification decisions, reviews,
evidence, runs, and reports. Conduit Desktop is the first graphical CGS client,
while `@conduit/runtime` provides the headless execution boundary.

Component versions are:

- Conduit Desktop `0.4.0`
- Conduit Runtime `0.4.0`
- Conduit Goal Specification `0.1.0`

This release preserves the existing goal-builder, implementation loop,
reviewer routing, evidence collection, persistence, report viewing and export,
provider integrations, and permission experience. It also adds a dedicated
Goal Analyst model setting, faster repository preparation, explicit legacy-data
migration, and the new Conduit application branding.
