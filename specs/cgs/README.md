# Conduit Goal Specification

CGS is Conduit's application-independent goal and verification format. The
current release is [CGS 0.1](./0.1/specification.md), shipped with the Conduit
0.4 release candidate and independently versioned as `0.1.0`.

CGS is being developed as Conduit's portable specification and reference data
contract. It is not represented as a standards-body or industry standard.
